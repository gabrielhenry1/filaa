Simulação de Filas: Encadeada e Circular

Fila Encadeada

Na fila encadeada, cada elemento aponta para o próximo, formando uma lista dinâmica (sem tamanho fixo).

Exemplo de Código (C)


typedef struct No {
    int valor;
    struct No* prox;
} No;

typedef struct {
    No* inicio;
    No* fim;
} Fila;

void enfileirar(Fila* f, int v) {
    No* novo = malloc(sizeof(No));
    novo->valor = v;
    novo->prox = NULL;
    if (f->fim) f->fim->prox = novo;
    else f->inicio = novo;
    f->fim = novo;
}

Explicação: Cada nó aponta para o próximo, e a fila cresce dinamicamente sem limite fixo.

Fila Circular

Na fila circular, o último elemento aponta para o primeiro, e o espaço é reaproveitado quando há remoções.

Exemplo de Código (C)


#define TAM 5
int fila[TAM];
int inicio = 0, fim = 0, qtd = 0;

void enfileirar(int v) {
    if (qtd < TAM) {
        fila[fim] = v;
        fim = (fim + 1) % TAM;
        qtd++;
    }
}

Explicação: Quando o fim chega ao final do vetor, ele volta para o início, formando um círculo.
